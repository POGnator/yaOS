# yaOS - the number 1 os out there

## Running
You'll need a Linux distro and emulator of some kind
- Install all required packages
- Run [build.sh](build.sh)
- Make a VM with yaOS.iso

> My system specs are:
- Windows 10/11
- Arch WSL
- qemu for Windows
- VSCode
